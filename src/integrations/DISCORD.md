# Discord Integration

## Available Events

- `message_create` - New message posted
- `message_update` - Message edited
- `message_delete` - Message deleted
- `message_reaction_add` - Reaction added to message
- `message_reaction_remove` - Reaction removed from message
- `presence_update` - User presence changed
- `thread_create` - Thread created
- `thread_update` - Thread updated

See https://discord.com/developers/docs/topics/gateway-events for details.
